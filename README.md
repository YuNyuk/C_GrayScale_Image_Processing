# C_GrayScale_Image_Processing

Visual Studio 로  개발한 GrayScale 이미지 처리 코드입니다.
Code 폴더 안에 전체 코드 압축파일이 있습니다.
압축 푸신 후 폴 내에 sln파일로 실행하시면 됩니다.
보고서 파일도 첨부합니다.

C언어를 기반으로 한 다양한 영상처리 기능을 활용한 <br>
GrayScale Image Processing <br><br>
